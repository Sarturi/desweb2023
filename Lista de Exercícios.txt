Lista de Exercícios

Instruções
Livro base Fundamentos de HTML5 e documentação oficial do HTML disponível em https://developer.mozilla.org/pt-BR/docs/Web/HTML. 

Exercício 1. Defina HTML?
Linguagem de marcação de texto.
Exercício 2. Defina o que são hipertextos.
São links que conectam páginas da web entre si.
Exercício 3. Defina o que são hiperligações.
São endereços web os quais se ligam a sites 
Exercício 4. Defina o que são hipermídias.
Produto multimidia ligado por hiperlinks
Exercício 5. Defina o que são linguagens de marcações.
Conjunto de marcadores para textos por exemplo o <i> itálico</i>
Exercício 6.  Defina o que são os browers
São os navegadores web como o chrome e operagx
Exercício 7.  Defina o conceito de renderização.
É o processamento para a combinação de material bruto.
Exercício 8.  Qual é o objetivo do HTML?
Permitir que um navegador leia e traduza marcações de texto.
Exercício 9.  Qual é o objetivo do CSS?
É fazer a estilização da página
Exercício 10.  Qual é a tag raíz de um documento em HTML?
<html>
Exercício 11.  Qual é o propósito da tag HEAD?
Fornecer metadados como a aceitação de caracteres especiais como o utf-8 e definir linguagem como lang= pt-br

Exercício 12.  Qual é o propósito da tag BODY?
São os dados visíveis pelo usuário como tabelas entre outros
Exercício 13.  Qual é a tag que representa um comentário em HTML?

Exercício 14.  Qual é a tag que representa um parágrafo?

Exercício 16.  Qual é a tag que configura o documento HTML para lidar com caracteres especiais?

Exercício 17.  Qual é a tag que representa um título/seção em um documento HTML? Quantos níveis possuem?
